Future Development

This roadmap outlines the future direction of the Kowalski project. Contributions are welcome!

Core

  • Long-term conversation storage
  • Conversation search and indexing
  • Context window management

Tools

  • Support for more document formats (DOCX, EPUB, HTML)
  • Image processing and OCR
  • Table extraction and processing
  • Academic paper processing (citations, figures, LaTeX)

Federation (Experimental)

  • Protocol selection (A2A, ACP, MCP, or custom)
  • Secure agent authentication
  • Persistent registry and orchestrator state
  • Federation-wide logging and monitoring

User Interface & Integration

  • Web interface
  • REST API
  • WebSocket support
  • Export conversations (PDF, HTML, Markdown)
  • Slack/Discord/Teams integration
  • Git/CI/CD integration
