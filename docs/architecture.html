High-Level Architecture

Core Components

The architecture is divided into several key layers:

  • Federation Layer: Manages communication and orchestration between agents.
  • Agent Layer: Contains the different types of specialized agents (Web, Academic, Code, Data).
  • Template Layer: Provides templates and configurations for building new agents.
  • Core Layer: The heart of Kowalski, containing the Tool API, Agent API, and other core components.
  • Tooling Layer: A collection of tools for interacting with various systems and data formats.
